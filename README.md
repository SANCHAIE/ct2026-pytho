## เกี่ยวกับหลักสูตร

หลักสูตรเริ่มต้นสำหรับผู้ที่ต้องการใช้ Python ในงาน Data Science และ AI โดยไม่ต้องมีพื้นฐานการเขียนโปรแกรมมาก่อน ครอบคลุมตั้งแต่ไวยากรณ์พื้นฐาน ไปจนถึงการวิเคราะห์ข้อมูลจริงและสร้างกราฟด้วย Python

**ระยะเวลา:** 2 ชั่วโมง | **ระดับ:** เริ่มต้น (Beginner)

## สิ่งที่จะได้รับ

- เขียน Python พื้นฐานได้อย่างถูกต้อง
- วิเคราะห์และแสดงผลข้อมูลด้วย `pandas` และ `matplotlib`
- ทำ Data Cleaning กับข้อมูลจริงที่มีค่าขาดหาย
- สร้างกราฟ Bar, Line, Scatter รายงานผลข้อมูลได้
